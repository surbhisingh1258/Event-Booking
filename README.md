# Event Booking System (MRRN Stack)

A full-stack Event Management web application built using the MERN stack that allows users to browse, book, and manage events, while admins can create and manage events efficiently.

Features:
1.User authentication & authorization (JWT-based)
2.Role-based access control (Admin & User)
3.Create, update, and delete events (Admin)
4.Browse and book events (User)
5.Booking management system
6.Responsive and user-friendly UI

Tech Stack
1.Frontend: React.js, HTML, CSS, JavaScript
2.Backend: Node.js, Express.js
3.Database: MongoDB
4.Authentication: JSON Web Token (JWT)
