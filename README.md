# Event Booking System (MRRN Stack)

A full-stack Event Management web application built using the MERN stack that allows users to browse, book, and manage events, while admins can create and manage events efficiently.

---

## Features

### Features
- User authentication & authorization (JWT-based)
- Browse available events
- Book events
- Manage bookings

### Admin Features
- Role-based access control (Admin & User)
- Create events
- Update events
- Delete events
- Manage all bookings


## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JSON Web Token (JWT)
